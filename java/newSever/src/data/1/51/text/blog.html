中新网6月24日电 据俄罗斯卫星网报道，“哈佛–哈里斯民调”为美国国会报“国会山庄报”所做的民调显示，美国总统特朗普的支持率长时间以来首次增长。
据报道，调研结果显示，48%的受访者支持特朗普在白宫主人位置上所做的工作，52%的美国人不支持他的工作。
上一次民调显示，特朗普的支持率为45%，同时不支持率为55%。
据悉，本次民调调查时间为6月19日至21日，2237名注册选民参与了此次调查。
此前，美国总统特朗普的支持率在5月中旬降到自1月20日就职以来的历史最低，仅为38%。
